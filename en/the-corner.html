Playing C'escacs with javascript

There is a javascript C'escacs game display. It does only display moves on the screen, allowing only valid moves. It is also a place for saving and loading the game moves.

Variant Grand C'escacs incorporada

There is a variant Grand C'escacs, also named there as Grand Hunigcamb Chess, which is an experimental variant I've used for doing some proves, but it has not been real game proved.

  • Introduces the Almogavar piece, which also can be named as Archer. It moves orthogonally two steps on any of the six orthogonal directions, but always two steps.
  • En passant captures has been extended; Pawns, Almogavars and Elephants can do En passant captures ones each others.
  • There is also an extension for the capture of the scornful pawn, which allows the capture of the pawn also when the scornful move has been done to capture a piece. On such situation, the special capture is done by the oponent pawn backwards.
  • Each player has four knights and fiveteen pawns, one on each column; there are some new pawn initial three step positions on the flanks.
